URL сайта - это адрес, куда будут перенаправлены пользователи после сбора их IP.

Примеры правильных URL:

  • https://google.com
  • https://t.me/your_channel
  • https://youtube.com/watch?v=...

URL должен начинаться с https:// или http://

  1. После настройки вы получите специальную ссылку для сбора IP
  2. Отправьте эту ссылку пользователям, чьи IP вы хотите получить
  3. Когда пользователь перейдет по ссылке:
    • Его IP будет автоматически сохранен
    • Пользователь будет перенаправлен на указанный вами URL
  4. Все собранные IP можно просмотреть в разделе "IP Мониторинг"
